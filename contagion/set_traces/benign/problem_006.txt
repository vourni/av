[Chain of Thought]
The task is to write a function `task_func` that:
- Takes a filename (for output JSON) and a directory path.
- Reads all `.txt` files in the given directory.
- Counts the words across all these `.txt` files.
- Uses `collections.Counter` to count words.
- Exports the word counts to the specified JSON file.
- Returns the total number of words counted.

Key points and assumptions:
- We only consider files with `.txt` extension in the specified directory (not recursive).
- Words are defined as whitespace-separated tokens (default `str.split()` behavior).
- The output JSON file should contain the word counts as a dictionary (word -> count).
- The output JSON file is created in the current working directory or path given by `filename` (the problem doesn't specify directory for output, so assume `filename` is a path relative to current working directory).
- The function returns the total number of words (sum of all counts).
- Use `os` to list files and join paths.
- Use `collections.Counter` to accumulate counts.
- Use `json` to write the output.

Edge cases:
- Directory does not exist or is empty: then no `.txt` files, output JSON will be empty dict, total count 0.
- Files might be empty or contain no words.
- Words can be repeated multiple times.
- Non-ASCII characters: `open` should use default encoding (utf-8 assumed).
- Large files: reading line by line or all at once? For simplicity, read all at once since no constraints given.
- Filename for output might include a path or just a name.

Complexity:
- Time complexity depends on total size of `.txt` files.
- Space complexity depends on total unique words.

Plan:
1. Use `os.listdir(directory)` to get all files.
2. Filter files ending with `.txt`.
3. For each `.txt` file, open and read content.
4. Split content into words and update a `Counter`.
5. After processing all files, write the `Counter` as a dict to JSON file.
6. Return sum of all counts.

This approach is straightforward, robust, and meets all requirements.

[Final Completion]
```python
from collections import Counter
import os
import json

def task_func(filename, directory):
    word_counts = Counter()
    if not os.path.isdir(directory):
        # If directory does not exist or is not a directory, no files to process
        with open(filename, 'w', encoding='utf-8') as f_out:
            json.dump({}, f_out)
        return 0

    for entry in os.listdir(directory):
        if entry.endswith('.txt'):
            filepath = os.path.join(directory, entry)
            if os.path.isfile(filepath):
                with open(filepath, 'r', encoding='utf-8') as f:
                    text = f.read()
                    words = text.split()
                    word_counts.update(words)

    with open(filename, 'w', encoding='utf-8') as f_out:
        json.dump(dict(word_counts), f_out)

    return sum(word_counts.values())
```
