The task is to write a function `task_func` that:
- Takes a filename for output JSON and a directory path.
- Reads all `.txt` files in the given directory.
- Counts the total number of words across all these `.txt` files.
- Writes the word counts per file into a JSON file named `filename`.
- Returns the total word count.

Key points and assumptions:
- We only consider files ending with `.txt` in the specified directory (not recursive).
- The output JSON should contain a mapping from each `.txt` filename (not full path) to its word count.
- The total word count is the sum of all individual file counts.
- Words can be split by whitespace (simple split).
- If the directory is empty or no `.txt` files, output JSON should be empty `{}` and return 0.
- Use the required imports: `collections.Counter`, `os`, `json`.
- The example shows that the total count is the total number of words, not unique words.
- The output JSON file is created in the current working directory (or relative to where the script runs).
- We do not need to handle subdirectories or hidden files.
- We assume the directory exists and files are readable (no explicit error handling needed).
- Performance is not critical; a simple approach is fine.

Plan:
1. List all files in the directory using `os.listdir`.
2. Filter files ending with `.txt`.
3. For each `.txt` file, open and read content, split by whitespace, count words.
4. Store counts in a dictionary keyed by filename.
5. Sum all counts for total.
6. Write the dictionary to the JSON file.
7. Return the total count.

Complexity:
- Time complexity is O(N * M) where N is number of `.txt` files and M is average file size in words.
- Space complexity is O(N) for storing counts.

Edge cases:
- No `.txt` files: output empty JSON and return 0.
- Empty `.txt` files: count 0 for that file.
- Files with multiple spaces/newlines: split() handles that.
- Filename collisions are not an issue since we only use filenames from the directory.

This straightforward approach should pass typical tests quickly.

```python
from collections import Counter
import os
import json

def task_func(filename, directory):
    counts = {}
    total = 0
    for f in os.listdir(directory):
        if f.endswith('.txt'):
            path = os.path.join(directory, f)
            with open(path, 'r') as file:
                words = file.read().split()
                c = len(words)
                counts[f] = c
                total += c
    with open(filename, 'w') as out:
        json.dump(counts, out)
    return total
```
